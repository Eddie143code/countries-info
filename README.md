# countries-info

The same app made in 3 different frameworks.

Angular app:
1. cd into ./angular-app
2. npm install, to install packages
3. ng serve, to run the app

React-app:
1. cd into ./react-app
2. npm install, to install packages
3. npm start, to run the app

Vue-app:
1. cd into ./vue-app
2. npm install, to install packages
3. npm run dev, to run the app
